# Fast Reader

A minimalist, high-performance RSVP (Rapid Serial Visual Presentation) reading extension for Google Chrome. Fast Reader helps you read text significantly faster by displaying words one by one at a fixed focal point, eliminating the need for eye movements (saccades).

## Features

- **RSVP Core Engine**: Accurate timing with dynamic delays for punctuation and long words.
- **Context Awareness**: "Continue Reading" feature intelligently finds and appends the next paragraph.
- **Focus Mode**: Clean overlay with zero distractions.
- **Theme Presets**: 
  - 6 Optimized themes: Dark, Light, Sepia, Night, Ocean, Forest.
  - Custom UI accents for each theme.
- **Advanced Controls**:
  - Variable WPM (Words Per Minute) speed control.
  - "Read All" mode to extract and read entire articles.
  - Built-in text editor to modify or paste custom text.
  - History trail and upcoming word preview.
- **Visual Customization**: Adjustable font size, font family, ORP (Optimal Recognition Point) highlighting, and element opacity.

## Installation

This extension is built for the Chrome browser.

1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked**.
5. Select the directory containing this repository.

## Usage

### Basic Reading
1. Select any text on a webpage.
2. Right-click and choose **"Read with Fast Reader"**.
3. **Alternatively**: Use the keyboard shortcut `Alt + R` (configurable).

### Custom Text
1. Click the extension icon in the toolbar.
2. Click **"Write Your Own Text"** to open a blank reader.
3. Paste or type your content and hit play.

### Controls
- **Space**: Play / Pause
- **Right / Left Arrow**: Seek forward / backward
- **Up / Down Arrow**: Increase / Decrease speed (WPM)
- **Esc**: Close reader
